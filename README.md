# 🏥 Chatbot de Saúde - Avaliação 01

Projeto desenvolvido para a disciplina de **Introdução à Computação** (1º Semestre). O objetivo é solucionar gargalos no atendimento hospitalar administrativo utilizando automação via WhatsApp.

## 🚀 Funcionalidades (MVP v1.0)
- **Agendar Consulta:** Fluxo interativo para marcação de especialidades e horários.
- **Cancelar Consulta:** Gestão autônoma de cancelamentos para redução de absenteísmo.
- **Ver Minhas Consultas:** Recuperação rápida de agendamentos confirmados.
- **Informações do Hospital:** FAQ sobre localização, contatos e funcionamento.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Python 3.x
- **Interface:** WhatsApp API (Simulação via terminal/web)
- **Lógica:** Estruturas condicionais, loops de repetição e dicionários para simulação de banco de dados.

## 📂 Como rodar o projeto
1. Clone o repositório: `git clone https://github.com/mourazks/NOME-DO-SEU-REPO`
2. Certifique-se de ter o Python 3 instalado.
3. Execute o arquivo principal: `python main.py` (ou o nome do seu arquivo .py)
